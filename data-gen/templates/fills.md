# フィル素材（1小節 / 4/4）

フィルは主に Snare/Toms/Cymbals を使う。下は **1小節分**。短いフィル（2拍）は前半をグルーヴ、
後半をフィルにして合計4拍にする。鳴らないトラックは `r` で埋める。

## tom-descend-16 — 16分タム下降（1小節フル）

```
Kick:    r.2 r.4 Kick.4 |
Snare:   Snare.16 Snare.16 Snare.16 Snare.16 r.2 r.4 |
HiHat:   r.1 |
Toms:    r.4 HiTom.16 HiTom.16 MidTom.16 MidTom.16 FloorTom.16 FloorTom.16 FloorTom.16 FloorTom.16 r.4 |
Cymbals: r.2 r.4 Crash.4 |
```

## snare-buildup-16 — スネア16分の煽り（1小節フル）

```
Kick:    r.2 r.2 |
Snare:   Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 Snare.16 |
HiHat:   r.1 |
Toms:    r.1 |
Cymbals: r.1 |
```

## half-bar-fill — 後半2拍フィル（前半は8ビート）

```
Kick:    Kick.4 r.4 r.2 |
Snare:   r.4 Snare.4 Snare.8 Snare.8 r.4 |
HiHat:   HH.8 HH.8 HH.8 HH.8 r.2 |
Toms:    r.2 HiTom.8 MidTom.8 FloorTom.8 FloorTom.8 |
Cymbals: r.2 r.4 Crash.4 |
```

## crash-accent — 締めのクラッシュ（1拍目アクセント）

```
Kick:    Kick.4 r.4 r.2 |
Snare:   r.1 |
HiHat:   r.1 |
Toms:    r.1 |
Cymbals: Crash.4 r.4 r.2 |
```
